**About**

This project is a digital glasswork management application.

**Prerequisites:** 

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`

**Run the app**

Run the app: `npm run dev`

**Build**

Build for production: `npm run build`

**Note**

This application previously used Base44 for backend services. All Base44 integrations have been removed. You will need to implement your own API endpoints to replace the stub functions in `src/api/api.js`.
